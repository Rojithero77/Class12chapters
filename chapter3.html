Chapter 3: Working with Functions

1. Define Function

A function is a block of code designed to perform a particular task. It runs only when it is called and may take inputs (parameters) and return outputs.

2. Syntax of Functions

return_type function_name(parameters) {
    // body of the function
}

Example in C++:

int add(int a, int b) {
    return a + b;
}

3. Types of Functions

  • Library Functions: Predefined functions provided by programming languages, like printf(), sqrt(), strlen().
  • User Defined Functions: Created by the programmer to perform custom tasks like calculateArea() or checkPrime().

4. Components of a Function

i. Function Prototype

A declaration of the function that tells the compiler about the function’s name, return type, and parameters before its actual definition.

int add(int, int);

ii. Function Call

When we want to execute the function, we call it using its name and pass required parameters.

int sum = add(5, 3);

iii. Function Definition

The actual block of code that contains the logic of the function.

int add(int a, int b) {
    return a + b;
}

iv. Return Type

The type of value the function returns. It can be int, float, void, etc.

void showMessage() {
    cout << "Welcome to DBMS Portal!";
}

2. Categories of Functions

i. Function with Return Type but No Arguments

This type of function returns a value but takes no arguments. It is useful when the logic doesn't depend on user input but still gives a result.

#include <stdio.h>

int getNumber() {
    return 10;
}

int main() {
    int x = getNumber();
    printf("Returned Number: %d\n", x);
    printf("Executed by Rojit Khadka\n");
    return 0;
}

ii. Function with Return Type and with Arguments

This function accepts input arguments and returns a value. It is most commonly used when both input and output are required.

#include <stdio.h>

int multiply(int a, int b) {
    return a * b;
}

int main() {
    int result = multiply(6, 7);
    printf("Multiplication Result: %d\n", result);
    printf("Executed by Rojit Khadka\n");
    return 0;
}

iii. Function with No Return Type and No Arguments

This function neither receives any arguments nor returns a value. It just performs a specific task, like displaying a message.

#include <stdio.h>

void message() {
    printf("Welcome to DBMS Chapter 3\n");
}

int main() {
    message();
    printf("Executed by Rojit Khadka\n");
    return 0;
}

iv. Function with No Return Type but with Arguments

This function takes arguments but does not return a value. It is useful when an action needs to be performed using external input.

#include <stdio.h>

void greet(char name[]) {
    printf("Hello, %s!\n", name);
}

int main() {
    greet("Sumi");
    printf("Executed by Rojit Khadka\n");
    return 0;
}

3. Storage Classes

Storage classes in C determine a variable’s scope (where it can be accessed), lifetime (how long it exists in memory), default initial value, and storage location. There are four main storage classes in C programming:

i. Automatic Storage Class (auto)

The auto keyword defines local variables with automatic storage. These are created when a function is called and destroyed when it ends.

  • Scope: Local to the function/block
  • Lifetime: Till function execution
  • Default Value: Garbage (undefined)
  • Storage Location: Memory (RAM)
#include <stdio.h>

void show() {
    auto int x = 10;
    printf("Value of x: %d\\n", x);
}

ii. External Storage Class (extern)

extern is used to declare a global variable or function that is defined elsewhere. It tells the compiler the variable exists, but memory will be allocated somewhere else.

  • Scope: Global (all files)
  • Lifetime: Entire program
  • Default Value: 0
  • Storage Location: Memory (RAM)
#include <stdio.h>

extern int a;

int main() {
    printf("Value of a: %d\\n", a);
    return 0;
}

int a = 25;

iii. Register Storage Class (register)

register requests the compiler to store the variable in a CPU register instead of RAM for faster access. Cannot take address using & operator.

  • Scope: Local
  • Lifetime: Till block exits
  • Default Value: Garbage
  • Storage Location: CPU Register (if available)
#include <stdio.h>

int main() {
    register int speed = 100;
    printf("Speed: %d km/h\\n", speed);
    return 0;
}

iv. Static Storage Class (static)

A static variable retains its value between multiple function calls. It is initialized only once and not re-created each time the function runs.

  • Scope: Local to block
  • Lifetime: Entire program
  • Default Value: 0
  • Storage Location: Memory (RAM)
#include <stdio.h>

void counter() {
    static int count = 0;
    count++;
    printf("Counter: %d\\n", count);
}

int main() {
    counter();
    counter();
    counter();
    return 0;
}

4. Recursive Function

A recursive function is a function that calls itself to solve a smaller sub-problem of the main task.

Syntax

return_type function_name() {
    if (base_condition)
        return result;
    else
        return function_name(); // recursive call
}

Example: Factorial using Recursion

#include <stdio.h>

int factorial(int n) {
    if (n == 0)
        return 1;
    else
        return n * factorial(n - 1);
}

int main() {
    int result = factorial(5);
    printf("Factorial: %d \n", result);
    printf("Executed by Rojit Khadka \n");
    return 0;
}

5. Passing Array to a Function

Arrays can be passed to a function by passing the array name (which acts as a pointer).

Example: Print Elements of Array

#include <stdio.h>

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

int main() {
    int nums[] = {1, 2, 3, 4, 5};
    printArray(nums, 5);
    printf("\\nExecuted by Rojit Khadka\\n");
    return 0;
}
